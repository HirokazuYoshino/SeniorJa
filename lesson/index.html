道　場

ホーム ：　この道場は、JavaScriptを始めて触れる初心者の道場です。

各lessonで、JavaScriptのプログラムを変更・追加・作成しながら、学んで行きます。

　lesson1 ・・・・ code*.js / 基本演算・条件分岐・繰返し・関数
　lesson2 ・・・・ 2code*.js/ DOM操作・イベント
　lesson3 ・・・・ 3code*.js/ スライドショー
　lesson4 ・・・・ 4code*.js/ jQuery
　lesson5 ・・・・ 5code*.js/ jQueryプラグイン
　lesson6 ・・・・ 6code*.js/ Web API

codeファイルに、JavaScriptをプログラムします。
lesson＊.html中に　script src="js\lesson*\*code.js　としてあります。
jsホルダーの中のcode＊jsの番号を削除して動かします。
動作テスト終了後は、番号を元に戻します。

VScodeを利用すると編集が便利です。
VScodeのLive Serverを使うと、編集後のホームページに直ぐに反映され、確認ができます。
